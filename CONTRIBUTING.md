# Contributing

Contributions to this project are welcome. In particular we are looking for test cases and flight test data for
model validation.

## Modifying existing models

When modifying an existing model, please merge such kind of pull requests to the **dev** branch.
Make sure to rebase the **dev** branch to main before creating a PR, based on the **dev** branch.

The dev branch can then be used to do extensive testing, updating of the examples (if needed) and tuning of the
controller parameters.

When all works well, create a pull request for the main branch.
